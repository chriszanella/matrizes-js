# 🧮 Matrizes com JavaScript

[![GitHub Pages](https://img.shields.io/badge/View%20Online-%F0%9F%9A%80-blue?style=for-the-badge)](https://chriszanella.github.io/matrizes-js/)

Esse é um projeto feito por mim, Christian Zanella, com o objetivo de **treinar lógica de programação, JavaScript puro, e TailwindCSS**, ao mesmo tempo que reforço os conteúdos de **matrizes que estou vendo na aula de matemática**.

## ✨ Funcionalidades

- Criação de matrizes com número de linhas e colunas personalizáveis
- Preenchimento automático com base em expressões envolvendo `i` e `j`
- Interface responsiva com **TailwindCSS**
- Tudo 100% rodando no frontend com JavaScript puro

---

## 💡 Tecnologias usadas

- HTML
- JavaScript (Vanilla)
- TailwindCSS

---

## 🎯 Objetivos do projeto

- Praticar lógica e estruturação de código JS
- Aprender Tailwind na prática
- Reforçar o conteúdo de matrizes da escola usando código
